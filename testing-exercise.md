# Testing Exercise

This exercise introduces unit testing using a simple calculator.

**Starter repo:** https://github.com/sefop/sefop-training-testing

## What you will find in the repo

- `calculator.py` — `add` is implemented as an example; `multiply` and `divide` are stubs
- `test_calculator.py` — a passing test for `add` to show the pattern
- CI pre-configured to run on every push and pull request (Python 3.12, pytest)

## Your task

1. Fork the repo and clone it locally
2. Run `pytest` — all tests should pass
3. Write a test for `multiply`, watch it fail
4. Implement `multiply`, watch it pass
5. Repeat for `divide`
6. Open a pull request using the provided template
